# Date Project

A React application built with Vite for fast development and modern frontend tooling.

## Project Overview

This project includes:

- React 18 with functional components
- Vite for fast development builds and hot module replacement
- Tailwind CSS for styling
- ESLint for code quality

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Project Structure

- `index.html` - application entry HTML
- `src/main.jsx` - React app bootstrap
- `src/App.jsx` - main application component
- `src/index.css` - global styles
- `src/App.css` - component-specific styles
- `vite.config.js` - Vite configuration
- `tailwind.config.js` - Tailwind configuration
- `postcss.config.js` - PostCSS configuration

## Notes

- This project uses the Vite React template.
- Customize the app by editing `src/App.jsx` and adding new components under `src/`.
- Keep `node_modules/`, build output, and editor settings out of version control using `.gitignore`.
